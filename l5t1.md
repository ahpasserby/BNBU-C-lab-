# Task No. L5T1

## Task Requirements
Write a program according to the following control flow diagram and output the final value of `n`.

### 控制流程图

```mermaid
flowchart TD
    A([开始]) --> B[/读取整数 n/]
    B --> C{n < 0?}
    C -->|是| D[n = -n]
    D --> E[n = 2 * n]
    C -->|否| E
    E --> F[/输出n的最终值/]
    F --> G([结束])

None
