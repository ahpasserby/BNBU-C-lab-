# Task No. A1T3

## Task Requirements
Write a program to read the amount of Euro dollars and change to HK dollar or RMB. Please output the amount of HK dollar or RMB depending on the choice. Assume the exchange rate between Euro dollar and HK dollar is `8.1`, between Euro dollar and RMB is `7.2`. Please choose proper data type and decimal places. Assume the input is always valid.

## Non-Functional Requirements

None

## I/O samples

Sample 1
```
Please input amount in Euro:11.5
Please choose the currency converting to (1: HK dollar, 2: RMB): 2
11.50 euro=82.80 RMB
Press any key to continue ...
```

Sample 2
```
Please input amount in Euro:11.5
Please choose the currency converting to (1: HK dollar, 2: RMB): 1
11.50 euro=93.15 HK dollar
Press any key to continue ...
```
