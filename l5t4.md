# Task No. L5T4

## Task Requirements
Write a program to implement the following control flow diagram. Assumption: the inputted height and weight are all bigger than 0.

```mermaid
flowchart TD
    A([Start]) --> B[/Read height (cm) and weight (kilo)/]
    B --> C{(Height - 105) >= Weight + 20?}
    C -->|Yes| D[/Output &quot;You are too slim&quot;/]
    D --> E([End])
    C -->|No| F{Weight - 10 < (height - 105) <= weigth + 20?}
    F -->|Yes| G[/Output &quot;You are fit&quot;/]
    G --> E
    F -->|No| H[/Output &quot;You are too fat&quot;/]
    H --> E
```
